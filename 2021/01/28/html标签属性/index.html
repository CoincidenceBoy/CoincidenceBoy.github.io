html属性

通用属性

  1. id : 元素唯一的id
  2. class : 元素的类名 可以有多个 例如 < p class=”class1 class2”>段落< /p >
  3. style : 规定元素的行内样式
  4. title : 鼠标停留在元素上时的提示文本 .

自定义属性

常用于 图片懒加载

html5 规定 自定义属性的格式为 data-* ,例如data-src
